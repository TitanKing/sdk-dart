scrollSpecifications method

Future<Map<String, dynamic>> scrollSpecifications (String index String collection String scrollId String scroll)

scroll is Time to live. It resets the cursor

  • d - days
  • h - hours
  • m - minutes
  • s - seconds
  • ms - milliseconds
  • micros - microseconds
  • nanos - nanoseconds

like: 2d for 2days

https://www.elastic.co/guide/en/elasticsearch/reference/5.4/common-options.html#time-units TODO: remove index, and collection if not needed

Implementation

Future<Map<String, dynamic>> scrollSpecifications(
  String index,
  String collection,
  String scrollId,
  String scroll,
) async {
  final response = await kuzzle.query(KuzzleRequest(
    controller: name,
    action: 'scrollSpecifications',
    scrollId: scrollId,
    scroll: scroll,
    index: index,
    collection: collection,
  ));

  return response.result as Map<String, dynamic>;
}
